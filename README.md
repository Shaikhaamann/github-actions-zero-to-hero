# 🚀 GitHub Actions Zero to Hero

This repository documents my learning journey with **GitHub Actions** as part of the **#90DaysOfDevOps** challenge.  
The goal of this project is to understand how CI/CD pipelines work and how automation can be used to build and deploy applications.

---

## 📌 Project Overview

In this project, I explored the fundamentals of **GitHub Actions** and implemented a workflow to automatically build and deploy my portfolio website.

The automation pipeline triggers when code is pushed to the repository and performs the deployment automatically.

## ⚙️ CI/CD Pipeline

This project demonstrates a simple **CI/CD pipeline using GitHub Actions**.

### Workflow Steps

1. Developer pushes code to GitHub
2. GitHub Actions workflow is triggered
3. The workflow builds the project
4. The application is deployed automatically
